What is Technova?

Department of Computer Science, Fakir Chand College presents Technova for the Year 2020 to be held from 22nd February,2020 to 29th February, 2020. It is our 2nd Version of Technova : The Inter College Technical Fest. This year we are here with more events. Visit our Events Section to get details.

  2. Date: 22nd - 29th February, 2020

Our Events

  • Treasure Hunt

    Find answers of some questions based on the selected topic. It is a One-week contest.
    Selected Topic: Adventure of Tintin.

  • Software Prototype Design

    Design a Desktop Software or Android app or Web App or any platform independent app prototype based on any one of the selected Theme.
    Selected theme: Library Management, Store Management

  • Logo, Banner, Poster Design

    Design a logo or banner or poster based on a given theme.
    Theme: Technical Event

  • Code Combat

    Different types Coding Contest will be organised in this year.
    Types:

    1. 1. Real Life Coding Contest: Question based on real life problems.
    2. 2. Subjective Coding: Problems from Data Structure and Algorithm.
    3. 3. Bug Hunt: Find bugs in given problem and correctify them.
    4. 4. Syntax Analyzer: Find the Programming Language from the gievn code snippet.

  • Arduino Coder

    Solve a problem through Arduino IDE and provided hardwares.
    Problem Statement: Will be provided on event day.

  • Mastering ICs

    Design a Circuit Diagram to solve the given problem only using 74 series ICs and Show result using provided Hardware.
    Problem Statement: Will be provided on event day.

  • Decrypt

    Decrypt a given encrypted Text Document using any encryption technology
    problem Statement: Will be provided on event day.

  • Database Developer

    Solve a query based on given table and data based on SQL.
    problem Statement: Will be provided on event day.

  • Mindbend

    Quiz Contest. It will be of two Different types.
    Types:

    1. 1. Online Mode: Online Quiz Contest based on Given Topics. Topics will be provided online before one week from the contest date.
    2. 2. Offline Mode: Offline Mode will cover all possible aspects of Technology.

  • Game Zone

    This year we managed to organize some Offline and Online Games. All more Details will be available soon.
